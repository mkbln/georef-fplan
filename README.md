# Digitaler Feuerwehrplan

Das Projekt enthält die relevaten Daten für den Testlauf zum digitalen Feuerwehrplan auf Basis von QGIS 3.4 LTR

Wichtige Grundkarten und weitere Quellen können unter https://gdi.berlin.de/viewer/main/ bezogen werden. 

Die Planunterlage für das Land Berlin ist als WMS Darstellungsdienst unter https://gdi.berlin.de/services/wms/plu verfügbar.
